Data Analytics¶

Building on foundational essential skills this course covers the main components of modern data analytics. Data visualization, database programming (SQL and noSQL), scientific investigation and technical communication are areas that are highlighted during this training. Students will become proficient using A/B testing methodologies and linear models. Building on a linear model foundation students will be introduced to commonly used predictive modeling and machine learning techniques.
